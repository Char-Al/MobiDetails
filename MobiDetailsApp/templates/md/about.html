How to use it

{{ info("MobiDetails is a DNA variant online interpretation tool. Users create the variants which must be linked to a gene.
To create a variant:
  • go to your gene page e.g. using the search engine at the bottom of each page (HGNC names),
  • then use the variant creation form by clicking on the 'create variant' button (substitutions or small indels HGVS c. nomenclature)
  • if the creation is successfull, a new link appears in the middle of the page,
  • which will bring you to the variant page. You're done!

For genes including multiple isoforms, you can select your isoform of interest. But beware, MobiDetails includes ons isoform which is considered as canonical (from UCSC) and will try to enforce the variant creation on that isoform (if so, you will be noticed). If this is not possible, it will try to create the variant on the selected isoform.

Once created, a variant will be stored in the system and you won't have to create it again.

You can also create an account (top right of all pages, icon ), which will allow you to quickly retrieve all the variants you created and all your favourite variants.
You can mark a variant as favourite in the variant page when you're logged in () by cliking on the star at the bottom of the left menu. You can retrieve your variants using the 'identity card' on the top right menu when you're logged in.
Registered users can in addition add an ACMG class to the variants and also contact each others via the application.

") }} {{ info("Search engine usage:
  • by exact HGVS c. or p. (allowing variations), e.g. 'p.Arg34Ter', 'p.Arg34*', 'c.100C>T', 'c.100c>t'...
  • by exact HGVS genomic nomenclature (hg19 or hg38), e.g. 'chr1:g.216595579G>A' or 'NC_000001.10:g.216595579G>A'
  • a partial number match on HGVS c. or p. variant names, e.g. '229' would return all variants with 229 in c. name or p. name
  • you can also use a decorator '%' with numbers to look for exact matches, e.g. '%2299' would return variants that contain 2299 in c. name or p. name, but not 22991, for example
  • gene symbol, e.g. 'USH2A'
") }}
